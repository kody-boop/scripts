#  个人自用脚本集合

本仓库用于存放个人日常使用的各类自动化脚本，涵盖数据备份、系统工具、效率提升等功能，目前主要包含思源笔记自动备份工具和发票申请自动化工具，后续将逐步添加更多实用脚本。

## 目录结构

```
scripts/
├── siyuan_backup/          # 思源笔记自动备份工具
│   ├── siyuan_backup.py    # 核心备份脚本
│   ├── run_backup.bat      # Windows批处理运行脚本
│   └── run_backup.vbs      # Windows无窗口运行脚本
└── fapiao/                 # 发票申请自动化工具
    └── src/                # 源代码目录
        ├── main.py         # 命令行版本主程序
        ├── gui_main.py     # 图形界面版本主程序
        └── read_excel.py   # Excel文件读取模块
```

## 脚本说明

### 1. 思源笔记自动备份工具（siyuan_backup）

#### 功能介绍
实现思源笔记（SiYuan）数据的自动备份、压缩并上传至腾讯云对象存储（COS），支持多平台运行，包含完整的日志记录和临时文件清理机制。

#### 核心特性
- 跨平台支持（Windows、Linux、macOS）
- 自动定位思源笔记默认数据目录
- 生成带时间戳的ZIP备份包
- 上传至腾讯云COS并支持失败重试（最多3次）
- 按天轮转日志并保留30天记录
- 自动清理临时文件，不占用额外存储空间

#### 使用说明
详见 [siyuan_backup 工具说明](siyuan_backup/README.md)

### 2. 发票申请自动化工具（fapiao）

#### 功能介绍
实现从Excel文件中读取发票申请数据，通过Selenium自动操作浏览器，在CRM系统中自动填写并提交发票申请的工具。提供图形界面和命令行两种使用方式。

#### 核心特性
- Excel数据自动读取
- 浏览器自动化操作（支持Chrome）
- 双操作模式（图形界面和命令行）
- 完善的日志记录和错误处理机制
- 跨平台支持（Windows和Linux）

#### 使用说明
详见 [fapiao 工具说明](fapiao/README.md)

## 通用使用注意事项

1. 部分脚本可能需要特定依赖环境，请根据各脚本说明安装所需依赖
2. 涉及敏感信息（如API密钥、路径配置等）的脚本，使用前请务必根据自身环境修改配置
3. 建议在首次使用脚本时，先手动测试运行，确认功能正常后再配置自动化任务
4. 脚本均为个人自用场景开发，可能需要根据不同系统环境进行适当调整
